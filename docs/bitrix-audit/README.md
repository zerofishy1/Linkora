# Bitrix Audit

Здесь хранятся артефакты живого исследования Bitrix24-портала:

- `screenshots/` — снимки ключевых разделов;
- `html/` — сохранённые HTML-версии страниц;
- `manifest.json` — краткая сводка по маршрутам, заголовкам и фрагментам текста;
- `click-map.json` — карта кликабельных элементов и переходов между разделами;

Скрипт запускается так:

```bash
BITRIX_LOGIN=... BITRIX_PASSWORD=... node ./tools/bitrix-audit.mjs
```

Дополнительный разбор кликов по сохранённому HTML:

```bash
node ./tools/bitrix-click-map.mjs
```

Чувствительные данные в проект не сохраняются. Логин и пароль передаются только через env, а cookies не пишутся в workspace.
